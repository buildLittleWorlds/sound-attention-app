# Same Note, Different Surroundings

This is a small one-page “paper app” about musical context.

The app plays the same pitch, **C**, in three different surroundings:

1. **Alone** — the note by itself.
2. **Inside a chord** — the same note held within a warmer harmonic context.
3. **As an ending** — the same note used as the final landing point of a short phrase.

The question behind the app is:

> Is the same note still the same when its surroundings change?

The app does not try to prove a theory of music. It gives the listener a simple, repeatable way to hear how context can change the role a sound seems to play. A pitch can feel bare, supported, tense, resolved, unfinished, or complete depending on what comes before it or around it.

## How It Works

This project is a single self-contained `index.html` file using:

- HTML
- CSS
- JavaScript
- WebAudio

There are no external libraries, no build tools, and no API keys.

## How to Use It

Open the page and press the three buttons.

Listen for what changes even though the central note stays the same.

## Publishing

This app is designed to work on GitHub Pages. Upload the `index.html` file to a GitHub repository and enable GitHub Pages for the repository.
